# Proyecto de Integración de API con Python

Este proyecto muestra cómo obtener datos de una API pública, procesarlos y visualizarlos en la terminal usando Python y pandas.

## Tecnologías utilizadas

- Python 3.12
- Pandas
- urllib.request (módulo estándar de Python)
- JSON (módulo estándar de Python)

Este proyecto **no requiere Excel** ni módulos externos para generar archivos, solo se muestra la información directamente en la terminal.

## Funcionalidad

1. Conecta con la API pública [JSONPlaceholder](https://jsonplaceholder.typicode.com/users) para obtener datos de usuarios ficticios.
2. Procesa la información para extraer:
   - ID
   - Nombre
   - Email
   - Ciudad
   - Empresa

3. Muestra los datos en forma de tabla en la terminal.
